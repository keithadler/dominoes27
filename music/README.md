# Background Music

Download royalty-free looping tracks and place them in this folder.

## Recommended Sources
- [Pixabay Music](https://pixabay.com/music/search/jazz%20lounge/) — Free, no attribution required
- [Mixkit](https://mixkit.co/free-stock-music/) — Free for commercial use
- [OpenGameArt](https://opengameart.org/art-search-advanced?field_art_type_tid%5B%5D=12) — Free game music

## Required Files

Download and rename to these filenames:

1. **chill.mp3** — Relaxed jazz/lounge track for normal gameplay (~2 min loop)
   - Search: "jazz lounge", "chill background", "smooth jazz instrumental"
   
2. **intense.mp3** — Upbeat/tense track for close games (~2 min loop)
   - Search: "upbeat jazz", "tension building", "exciting background"

3. **victory.mp3** — Short celebration jingle (~5-10 seconds)
   - Search: "victory fanfare", "win jingle", "celebration short"

## Suggested Pixabay Tracks
- Chill: "The Best Jazz Club In New Orleans" by PaoloArgento
- Intense: "Jazzy Abstract Beat" or any upbeat jazz
- Victory: Any short fanfare/celebration clip

The game will fall back to synthesized audio if these files are not found.
